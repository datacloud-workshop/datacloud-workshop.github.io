About the Workshop

The Salesforce Data Cloud workshop guide includes a self-guided hands-on lab and supplemental sections such as ideation questions to help in creating a Data Cloud reference architecture and resources with links to videos and implementation guides.

Your facilitator will provide you with login/password for a Data Cloud demo org for the hands-on lab.

This is not an official Salesforce guide and is shared here for demo only. Please refer to Salesforce.com for official documentation.
