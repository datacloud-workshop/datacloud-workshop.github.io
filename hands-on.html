Salesforce Data Cloud: Hands-on

Time required: 45 minutes

Workshop Goal: The Hands-Lab is designed to provide a first-hand experience of the tools and configuration tasks in the Salesforce Data Cloud platform.

Workshop Use Case Architecture

How to Complete This Workshop

The workshop is self-guided lab using an instance of Salesforce Data Cloud. Your instructor will provide the login for the Data Cloud intance.

This is an exploratory workshop. You will be able to click through and examine the configurations but you will not create new configurations.

Part - 1: Pre-configurations

This section has been completed for you by the instructor. This is only an informational section for participants.

Data Cloud has been provisioned in this instance of Salesforce and is available as an app in the Org. This Org is now known as the Home Org of Data Cloud.

"Turn on" Data Cloud by adding users to permisson sets. For example assign the Data Cloud Admin permission set to the sysadmin who will manage the Data Cloud instance.

Data Cloud Permissions

With the permission sets assigned, a user can view the Data Cloud app in the App Launcher.

Data Cloud App

You have been assigned the Data Cloud User permission set which allows you to examine the app and all the configuratins.

Data Cloud App - Menu Tabs

The Data Cloud Setup button takes you to the pre-configuration page.

Data Cloud App - Setup

Click on the CRM Connector and then Connect. The CRM Connector provides a pre-built connection from Salesforce Orgs to Data Cloud. You can set up ingestion of records (i.e., create Data Streams) from either standard or custom objects. The ingestion schedule is hourly refresh.

Note that even though Data Cloud is provisioned in the Home Org, you still need to explictly establish a connection to the Org which is the Home Org.

Connector Configuration

 

Install the Data Bundle for the CRM Connector. The Data Bundle consists of pre-defined mappings for certain Salesforce objects (Account, Contact, Leads for Sales Bundle and Account, Contact, Cases for Service Bundle).
